"O método utilizado de uma maneira geral é muito bom, mas acrescentaria alguns detalhes como:
1- Feedback semanal dos trabalhos semanais:
	1.1 - Opções de feedback: a) Correção e divulgação das notas de cada trabalho semanalmente; b) Apresentação de 1 ou 2 pessoas rapidamente no início de cada aula, com análise/correções do professor; c) Mostrar um modelo de trabalho ideal no início da aula.
2- Realização de 2 trabalhos semanais a mais que cobrassem Testes de Hipótese e Mapas, para forçar uma melhor preparação dos alunos para a Apresentação Final.  
3- Dispor previamente aos alunos os comandos usados em cada aula, para evitar perda de tempo com cópia braçal do quadro.
	3.1- Pode-se dispor apenas os comandos relativos a uma base de dados e durante a aula cobrar que os alunos repliquem em outra base de dados.
4- Sobre o Trabalho Final:
	4.1 - Fixar no topo do grupo do Facebook o post com a planilha do Google (para os alunos inserirem os dados e links dos respectivos trabalhos);
